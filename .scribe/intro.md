# Introduction



<aside>
    <strong>Base URL</strong>: <code>http://localhost:8000</code>
</aside>

    Bem-vindo à documentação da API do **Controle de Revisões de Veículos**.

    Esta API permite gerenciar usuários, veículos, marcas, pessoas e o histórico de revisões, oferecendo endpoints para autenticação, cadastro e consulta de dados de forma segura e integrada.

    <aside>Conforme você rola a página, verá exemplos de código para consumir a API em diferentes linguagens de programação na área escura à direita (ou junto ao conteúdo principal, no celular).
    Você pode trocar a linguagem usada nas abas no canto superior direito (ou pelo menu de navegação no topo, no celular).</aside>

    <aside>A maioria dos endpoints exige autenticação via **Bearer Token**, obtido através da rota de login. Consulte a seção "Autenticação" para mais detalhes.</aside>

    ## Como usar esta documentação

    - Navegue pelas seções no menu lateral para encontrar o endpoint desejado
    - Use o botão **"Try It Out"** para testar as requisições diretamente pelo navegador
    - Todos os exemplos de resposta são baseados em dados reais da aplicação

    Em caso de dúvidas ou problemas, entre em contato com o time de desenvolvimento.
